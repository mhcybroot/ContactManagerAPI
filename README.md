Title: Contact Manager API (REST API)
Problem Overview: 

The goal of this assignment is to develop a RESTful Contact Manager API using Spring Boot. The primary focus is on the Data Access Layer (Repository), where you must implement complex data retrieval operations that leverage specific Spring Data JPA features.

You will create and utilize a Contact Entity and a corresponding ContactRepository, implementing various methods to demonstrate proficiency with derived queries, custom JPQL, parameter passing, pagination, and result projection.

Task:

Create a Java Spring Boot application that provides RESTful endpoints for managing contacts.

Entity Structure(Book):

- Id (Long) Primary key, auto-generated

- First name

- Last name

- Phone no (String)

- E-mail

- Is Active

- Category (String) (work, school, family, client, etc.)

- Creation Date (LocalDateTime)
